Вход в режим программирования:
Подключает терминальную программу с параметрами порта: 115200 NParity NFlowControl 1stopbit.
При нажатой кнопке KEY1(PC5)(можно запрограммировать любые из KEY1...KEY6) включаем уст-во.
(К сожалению при задействовании порта B - невозможно подключиться к станции по USB 
- проверял на разных компиляторах)

Получаем сообщение: "ENTER PROGRAMMING MODE"
Далее можно программировать каналы по такой схеме:

G - групповой вызов
P - частный
N - количество каналов(1..6)
Y = номер канала
X - номер группы(абонента)

Формат сообщения- строка : "N YGX YPX YPX"
Например для 3-х каналов(групповой-11(1канал) и два приватных 22(3канал) и 33(2канал))  "3 1G11 3P22 2P33"
В случае успешной записи получаем сообщение "OK!" иначе  "ERROR!".
